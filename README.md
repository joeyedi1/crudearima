# ARIMA Time Series Forecasting

This project builds an ARIMA model to forecast time series data.

## Project Structure
- `data/`: Contains raw and processed data (not included in repo).
- `notebooks/`: Jupyter notebooks for analysis and modeling.

## Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage

1. Place your data CSV in `data/raw/`.
2. Open `notebooks/01_arima_model.ipynb`.
3. Follow the steps in the notebook to train and forecast.
